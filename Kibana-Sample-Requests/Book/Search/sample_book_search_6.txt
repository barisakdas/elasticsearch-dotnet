Belirli Bir Kelimeyi İçeren ve Belirli Bir Stok Miktarından Fazla Olan Makaleleri Arama:

GET /books/_search
{
  "query": {
    "bool": {
      "must": {
        "match": {
          "abstract": "robotik"
        }
      },
      "filter": {
        "range": {
          "stock": {
            "gt": 50
          }
        }
      }
    }
  }
}
