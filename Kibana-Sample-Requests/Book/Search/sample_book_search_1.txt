Belirli Bir Tarihten Sonra Yayınlanmış ve Belirli Kategorilerdeki Makaleleri Arama:

GET /books/_search
{
  "query": {
    "bool": {
      "must": {
        "range": {
          "publishdate": {
            "gte": "2024-01-01"
          }
        }
      },
      "filter": {
        "terms": {
          "categories.keyword": ["Teknoloji", "Bilim"]
        }
      }
    }
  }
}
